# gitの一連の流れを自動化するbashファイル

以下のgit.shを.gitと同階層に置いた上で実行

## git.sh
<!--```sh!-->
{% highlight shell linenos %}
#!/bin/sh

git add .
if [ $# -eq 0 ]; then
    git commit -m "Update"
else
    git commit -m "$1"
fi
git push
{% endhighlight %}
<!--```!-->

## 実行方法
コミットメッセージがUpdateで良い場合
```console
$ bash git.sh
```
コミットメッセージを指定する場合
```console
$ bash git.sh <commit message>
```
